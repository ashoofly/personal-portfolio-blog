A lot of the color / palette documentation is scattered, and also they are split into 'MUI System', 'MUI Core' etc.


Here is a comprehensive list of the default theme object: https://mui.com/material-ui/customization/default-theme/

Using default theme colors

To use your themed palette, for any MUI elments, you can set: https://mui.com/material-ui/customization/palette/
```
color="primary.contrastText"
backgroundColor="primary.dark"
```





Here are the colors you can use: https://mui.com/material-ui/customization/color/#color-palette
